Executing the Process

Kindly follow the order. If you may have an issue refers to FAQs
  1. Open Salary

    This is a process of opening salary for all active employees. This is a crucial process as it pre-determined which employees qualifies to receive the salary for that month.

  2. Generate Payroll preview

    This is a process of generating a salary preview, detailing all the deductions for each and every employee. This Payroll preview will be flexible in that it can allow the authorized person only to modify the entries for that month.

  3. Sent Preview Payroll Notification Mail Report

    Through this event, the executive will receive a generated reports of Previewayroll of all employees. It act as a preview of salary to be paid but still flexible for modifications.

  4. Close Payroll Generation

    This event will close all the payroll generation, hence no employee will have salary after it has been closed.

  5. Close and commit Salary

    This event is an important process. It will commit the salary and generate reports for the actual salary that will be sent to executives.

  6. Sent Actual Payroll Notification Mail Report

    THis process will sent a actuall report to executives, showing all the commited salary to be tranfer to bank for that particular month.

  7. Sent P9form and Payslip Reports Mail

    This event will sent detailed reports to all employees personal emails which includes the detailed payslip for that month and the P9 form for that particular year.
